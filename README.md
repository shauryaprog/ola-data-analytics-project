# Ola Data Analytics Project

## Project Overview
This project analyzes Ola ride-booking data using SQL and Power BI to extract business insights related to bookings, cancellations, revenue, and customer behavior.

## Tools Used
- SQL
- Power BI
- Excel

## Business Problems Solved
- Peak booking hour analysis
- Cancellation analysis
- Revenue analysis
- Top customers identification
- Payment trend analysis

## Key Insights
- Peak demand observed during evening hours
- Driver unavailability caused high cancellations
- UPI was the most used payment method
